Prime Bot | Gorktimus Prime

AI-powered meme coin intelligence and risk analysis terminal.

Overview

Prime Bot is a Telegram-based token scanner and risk scoring engine designed to analyze:

Liquidity levels

Volume activity

Holder concentration

Momentum spikes

Basic rug risk indicators

This project is part of the Gorktimus Prime ecosystem — focused on building infrastructure for smarter meme coin trading.

Features (MVP)

/scan — Pull token market data

/score — Generate risk score (0–100)

/watch — Add token to alert system

Vision

Data before impulse.
Intelligence before execution.
